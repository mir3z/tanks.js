# tanks.js

[Scorched Earth](https://en.wikipedia.org/wiki/Scorched_Earth_(video_game)) based game in JavaScript.

## Controlling
* PgUp, Arrow Up - increase power
* PgDown, Arrow Down - decrease power
* Arrow Left/Right - change angle
* Tab - switch weapon
* Enter - fire
